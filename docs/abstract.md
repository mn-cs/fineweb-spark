# Abstract

This project analyzes the FineWeb-Edu (Sample-10BT) dataset, a large-scale corpus containing approximately 9.6 million web-crawled documents annotated with educational quality scores. The dataset's decompressed size exceeds 100 GB, making it unsuitable for standard in-memory processing with tools such as Pandas and motivating the use of Apache Spark for distributed computation on the SDSC Expanse high-performance computing cluster.

The primary objective of this project is to build a scalable machine learning pipeline to classify web documents as having high or low educational value based on their linguistic characteristics. The proposed workflow includes an extract–transform–load (ETL) stage for text cleaning and tokenization, followed by feature extraction using TF-IDF vectorization implemented with Spark MLlib. Supervised learning models, including Logistic Regression and Random Forest classifiers, are trained and compared to evaluate their effectiveness in distinguishing high-quality educational content from low-quality web text.

This project demonstrates how distributed machine learning techniques can be applied to large text corpora to support scalable and reproducible educational content analysis.

## Dataset

**Source:** [FineWeb-Edu Sample-10BT](https://huggingface.co/datasets/HuggingFaceFW/fineweb-edu/viewer/sample-10BT) hosted by Hugging Face

The dataset is the "Sample-10BT" subset of FineWeb-Edu. It contains 10 billion tokens (approximately 9.67 million rows) of web text data stored in Parquet format. Key columns include:

| Column           | Type    | Description                                                                       |
| ---------------- | ------- | --------------------------------------------------------------------------------- |
| `text`           | string  | Raw article content                                                               |
| `token_count`    | integer | Number of tokens in the document                                                  |
| `language_score` | float   | Language confidence score                                                         |
| `int_score`      | integer | Educational quality score (ground truth, generated by a Llama-3-based classifier) |

**Compressed size on disk:** ~28 GB
